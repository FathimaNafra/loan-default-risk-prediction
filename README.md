# 🚀 Loan Default Risk Approval Prediction System

## 📌 Overview
This project is a Machine Learning-based system designed to predict **loan approval status** and generate a **loan risk score** using advanced classification and regression models. It also includes model explainability using SHAP and a Flask web application for real-time predictions.

---

## 🎯 Objectives
- Predict whether a loan will be approved or rejected
- Generate a continuous risk score for each applicant
- Improve decision-making using machine learning
- Provide explainability for model predictions

---

## 🧠 Machine Learning Approach

### 1️⃣ Classification Model
- Algorithm: LightGBM Classifier
- Output: Loan Approval (Yes/No)

### 2️⃣ Regression Model
- Algorithm: LightGBM Regressor
- Output: Loan Risk Score (continuous value)

---

## ⚙️ Features
- Data preprocessing and cleaning
- Handling missing values
- Feature engineering
- Model training using LightGBM
- Model evaluation using performance metrics
- SHAP-based explainability (feature importance)
- Flask web application for user interaction

---

## 🛠️ Tech Stack
- Python 🐍
- Scikit-learn
- LightGBM
- SHAP
- Flask
- Pandas / NumPy
- HTML / CSS (Frontend UI)

---

## 📊 Model Explainability
SHAP (SHapley Additive exPlanations) is used to:
- Understand feature contributions
- Improve model transparency
- Explain individual predictions

---

## 🌐 Web Application
A Flask-based web interface allows users to:
- Input applicant details
- Get instant loan approval prediction
- View loan risk score results
